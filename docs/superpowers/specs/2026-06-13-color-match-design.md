# Цвет-матч фигуры под сцену — спека

**Дата:** 2026-06-13
**Статус:** утверждён в чате («го цвет»)
**Родитель:** Lux-рендерер (`2026-06-12-lux-renderer-design.md` §4 конвейер
кадра). Матт и транспорт закрыты; это первый шаг «пинтерест»-интеграции
в `src/lux/compositor.ts`.

## 1. Проблема

Фигура снята вебкой при своём свете (часто холодный офисный), интерьер —
со своим (тёплый). Глаз читает рассинхрон цветовой температуры как «наклейка».
Статичный per-интерьер LUT этого не лечит (он один на всех посетителей,
не знает реальный свет фигуры). Нужна автоподстройка под фон в рантайме.

## 2. Решение (v1: баланс белого + экспозиция, не полный Reinhard)

1. **Средний цвет сцены** — GPU-редукция фона. У компонента уже есть
   `wrapRT_A` (1/4, блюр фона); добавляем мип-цепочку/добивочный даунсэмпл до
   ~1×1 → `uSceneMean` (vec3). Ноль чтений GPU→CPU (без сталлов).
2. **Применение в шейдере фигуры:** не заливаем фигуру средним (поседеет), а
   переносим только **цветовой каст и экспозицию** сцены. Из `uSceneMean`
   берём нормализованную хрому `mean / luma(mean)` и домножаем rgb фигуры на
   `mix(vec3(1.0), хрома, strength)`; экспозиция — `mix(1.0, luma(mean)/0.5,
   strengthExp)`. Сила мягкая (дефолт 0.25/0.15) — коррекция, не перекраска.
3. **Порядок:** после LUT, до light wrap (свет сцены ложится на фигуру, потом
   её край обнимается фоном). Дев-тогл (как F1–F4): `uColorMatchOn`.

Полный перенос (mean+std самой фигуры, Reinhard) — вне v1: нужна альфа-взвеш.
редукция фигуры (доп. пассы), а каст/экспозиция дают основной эффект.

## 3. Конфиг

`LUX_CONFIG.colorMatch = { cast: 0.25, exposure: 0.15 }` — крутится без
пересборки. Тогл в `devFlags` рядом с lut/wrap/shadow/grain.

## 4. Тестирование

TDD (web): чистая функция `colorMatchUniforms(sceneMeanRGB, cfg)` →
{ castMul:vec3, expMul:number } — нейтральный серый фон даёт castMul≈(1,1,1);
тёплый фон → castMul.r>castMul.b; strength=0 → единичные множители.
devFlag colorMatch (вкл/выкл). Шейдер — ручная/визуальная приёмка (прецедент
компонента). Существующие web-тесты зелёные.
Приёмка: live на интерьере спальни (тёплый) — фигура теплеет под свет комнаты,
toggle off возвращает «наклейку»; на балконе (холодный) — наоборот.
