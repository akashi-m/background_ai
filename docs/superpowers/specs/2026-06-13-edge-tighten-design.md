# Поджатие края альфы (убрать остаточную «дымку») — спека

**Дата:** 2026-06-13
**Статус:** утверждён в чате («1»)
**Родитель:** `2026-06-13-spill-suppression-design.md` (цвет края вычищен fgr;
осталась мягкая полупрозрачная полоса — ширина альфа-перехода RVM)

## 1. Где живёт решение

Поджатие края — **ручка рендерера**, не capture. Capture отдаёт сырую мягкую
альфу (источник истины): рендерер уже ремапит её `smoothstep(feather.x,
feather.y, a)` в шейдере фигуры, и light wrap (`a*(1-a)`) работает по этой же
полосе. Пережигать резкий край в capture нельзя — рендерер потеряет диапазон
для feather/wrap. Текущий дефолт `LUX_CONFIG.feather = [0.05, 0.95]` почти
линеен → полоса не сужается. Сузим окно (например `[0.35, 0.80]`): резкий, но
с 1–2 px сглаживания для анти-алиаса.

## 2. Проблема верификации и её решение

Заказчик оценивает матт по `/frame.png` (live-вид мылит WebRTC-кодек). Чтобы
видеть поджатие losslessly, `/frame.png` получает необязательный параметр
`?feather=lo,hi` — применяет ТУ ЖЕ smoothstep к альфе перед упаковкой PNG.
Хранимая/стримящаяся альфа остаётся сырой (неразрушающе); параметр — только
превью того, что сделает шейдер. Без параметра — текущее поведение (сырая альфа).

## 3. Объём

1. **capture:** `shape_alpha(alpha, lo, hi)` — чистая функция (clamp lo<hi,
   smoothstep); `/frame.png?feather=lo,hi` применяет её. Невалидный параметр →
   игнор (сырая альфа), не 400 (дев-инструмент).
2. **подбор:** на живом S24 перебрать пару окон через `/frame.png?feather=`,
   выбрать глазами.
3. **renderer:** записать выбранное окно в `LUX_CONFIG.feather`.

Вне скоупа: эрозия/морфология альфы (если smoothstep хватит — не усложняем),
живые дев-клавиши feather (вернёмся, если понадобится частая подстройка).

## 4. Тестирование

TDD (Python): `shape_alpha` — границы (lo→0, hi→1, середина монотонна),
невалидные lo≥hi → сырая альфа. `/frame.png?feather=0.35,0.8` → 200, PNG,
альфа-канал жёстче сырого (доля пикселей в 0.1..0.9 падает); битый параметр →
как без него. Web: значение `LUX_CONFIG.feather` в диапазоне (юнит конфига).
Приёмка: `/frame.png?feather=` на S24 — «дымка» по краю волос ушла, прядей не
теряем.
