# Гибридная тень: запечённая база (Blender) + живой слой

Дата: 2026-06-16
Статус: дизайн (брейншторм-валидация пройдена — юзер одобрил лук запечённой тени)

## Проблема

Процедурная/прокси-тень real-time (капсулы + multiply-blit + зерно/тинт) после многих
раундов тюнинга всё равно читается «ненатурально». Корень: (1) форма-кастер грубая
(капсулы ≠ тело), (2) real-time shadow maps жёсткие, (3) процедурное поверх борется с этим.
Потолок «подделки» достигнут.

## Подход (выбор юзера: A + живой слой)

Развести роли по двум слоям:

1. **База — запечённая в Blender тень** (offline, Cycles, реальный AREA-свет комнаты).
   Несёт реализм: настоящая мягкая полутень, чёткие контакты у стоп, трассировка/GI.
   Статична (поза «стоит»).
2. **Динамический слой — живой, бледный, сильно размытый.** Несёт *жизнь*: повторяет
   взмах руки/наклон. Качество НЕ важно — бледность+блюр прячут грубость; реализм уже
   дал базовый слой. Накладывается поверх базы с большей прозрачностью.

Ключевая идея: real-time слой больше не обязан быть красивым, только отзывчивым.

## Валидация (сделано через Blender MCP)

В сцене B2-10 уже есть: меш `Human 2` (Rex, ~2.2k полигонов), камера плейта
(18мм, fovY 1.176), ключевой свет `Key_Living_Warm` — **AREA** 0.6×0.25 @ energy 260,
Cycles, 1080×1920. Тестовый shadow-catcher рендер (`/tmp/shadow_only_floor.png`)
подтвердил: мягкая органичная тень с чёткими контактами у стоп — на порядок натуральнее
процедурной. Юзер одобрил лук.

## Архитектура

### Бейк-пайплайн (offline, Blender MCP, недеструктивно)

- Дубль `Human 2`, ЦЕНТР bbox → точка стойки (геометрия смещена от origin — позиционировать
  по bbox, не по origin!), ступни на z=0.
- Временный shadow-catcher план @ z=0, `is_shadow_catcher=True`.
- Прочие меши `hide_render=True` (только человек + catcher + лампы → чистая тень человека,
  без теней мебели на catcher).
- `film_transparent=True` → альфа = маска тени. Рендер из камеры плейта (перспектива уже
  верная) в полном кадре 1080×1920.
- Всё откатывается (try/finally: восстановить hide_render, render-настройки; удалить
  временные объекты). Юзер держит .blend сохранённым как бэкстоп.
- Выход: `assets/worlds/living/shadow_baked.png` — полнокадровая маска тени (альфа).

### Рантайм-композит (compositor.ts)

**Фаза 1 — база:** загрузить `shadow_baked.png`, наложить multiply на `compositeRT`
(как плейт — cover-fit), тёплый тон + регулируемая сила. Тень зафиксирована в экранных
координатах под точкой стойки киоска. Заменяет прокси-проводку как основную тень.

**Фаза 2 — живой слой:** поверх базы — бледный сильно размытый слой, привязан к ногам,
повторяет позу. Кандидаты:
- **наш прокси** (уже разведён в C.6, артикулирует от позы) — бледный+доп.блюр, 0 нового кода;
- **матт-проекция** (точная форма тела) — новый шейдер проекции вырезки на пол.
  → v1 берём прокси-бледный (переиспользование), матт — апгрейд если артикуляция слишком груба.

## Решения

- **Киоск-фиксированная база:** человек стоит на метке → база запечена для этой точки,
  фиксирована в кадре. Малые движения ловит живой слой. (Не «штамп по ногам» — проще и
  достаточно для киоска.)
- **Дин-слой v1 = прокси-бледный** (переиспользуем весь тюнинг как сублой).
- Фазирование: сначала база (большой выигрыш натуральности), оценка; потом живой слой.

## Риски

- Поза/позиция бейка ≠ реальная стойка → перспективный сдвиг. Митигейт: киоск-метка;
  при необходимости перезапечь на уточнённой точке.
- MCP-таймаут на полном рендере → finally не отработает, сцена грязная. Митигейт: умеренные
  сэмплы (160–256) + денойз + сохранённый .blend.
- Запечённая база статична (одна стойка) — это by design; артикуляцию даёт живой слой.

## Тестирование

- Бейк визуально оценивается в превью (наложение base-маски на плейт), приёмка юзером.
- Композит: юнит-зеркало multiply (как сейчас), tsc/vitest green.
- Живая приёмка на теле юзера (камера — из терминала юзера).

## Незакоммиченное

Прокси-тюнинг (5 раундов: слияние ног, шея/без головы, контактная лужа, смещения,
плотности, зерно-как-блоб) держится без коммита по просьбе юзера — становится кандидатом
в Фазу-2 (живой слой) или фолбэк. Зелёный (159 тестов, tsc чист).
